# Recordly Custom

这是基于 Recordly 调整后的录屏工具备份仓库，用来保存当前已测试通过的安装版应用内容和本地改造记录。

## 当前状态

- Windows 安装版已完成并测试可用。
- 支持录制时显示摄像头画面，并把摄像头画面合成进成片。
- 摄像头画面支持移动、缩放，并保存上一次的位置和大小。
- 麦克风等录制配置会保存最近一次选择。
- 录制条用于录制过程控制，但不会出现在导出成片里。
- 导出视频声音已验证正常。
- 自动字幕暂时不用，因此不需要下载 Whisper 模型。

## 目录说明

- `asar-inspect/`：当前安装版应用包解出的内容，便于备份和追踪已生效的改造。
- `release/`：本地构建/安装包缓存，已忽略，不提交到 Git。
- `recordly-slim-backup-*`：安装目录瘦身实验备份，已忽略，不提交到 Git。

## 注意

这个仓库当前更适合做“稳定版本备份”。它不是最理想的完整开发源码结构，因为当前可找到的是打包后的应用内容。如果后续找回原始 Recordly 源码目录，建议再单独建立标准源码仓库。

安装包建议通过 GitHub Release 附件发布，不直接提交到源码仓库。
